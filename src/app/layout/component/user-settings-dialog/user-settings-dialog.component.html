FAQ Q.: How do i open the search in my default browser?

A.: Holding ctrl while clicking will open the link in the default browser.

 Q.: What does the graph represent? What is mode/mean/median?

A.: The bar indicates the distribution of the item value. It basically groups the values by value and counts them.
The highest bar is the Mode. Also named 'most common value'. Mean is the average. Median is the centered value of all values.

 Q.: My screen turns black/white after toggling any feature. Why?

A.: This is mostly caused by an disabled Aero / DWM. Therefore try enabling this windows feature.
As it's required to create transparent windows.

 Q.: The graphics are blurry/ graphic bug?

A.: This is mostly caused by incompatible graphic drivers for the current chromium version.
Please run 'run-without-gpu.bat' in order to start the overlay without gpu support (hardware acceleration).
